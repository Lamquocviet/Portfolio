### **Quy Trình Phát Triển**

#### **1️⃣ Giai Đoạn Lập Kế Hoạch**
- Xác định cấu trúc trang web
- Phân chia thành các phần chính: Header, About, Skills, Projects, Certificates, Footer
- Lên kế hoạch responsive design cho các thiết bị khác nhau

#### **2️⃣ Giai Đoạn Thiết Kế**
- Tạo layout HTML semantic
- Thiết kế CSS modular theo từng section
- Sử dụng CSS Grid và Flexbox cho layout
- Xây dựng hệ thống animation smooth

#### **3️⃣ Giai Đoạn Phát Triển**
- **HTML Structure**: Tạo cấu trúc HTML5 semantically correct
- **CSS Styling**: Chia nhỏ CSS thành multiple files để dễ bảo trì
  - `main.css`: CSS global
  - `responsive.css`: Media queries cho mobile, tablet, desktop
  - `animation.css`: CSS animations
  - `sections/*.css`: CSS cho từng section riêng
- **JavaScript**: Thêm interactivity (nếu cần)
- **Assets**: Tối ưu hình ảnh, SVG icons

#### **4️⃣ Giai Đoạn Kiểm Thử**
- Test responsive trên các kích thước màn hình khác nhau
- Test trên các trình duyệt khác nhau (Chrome, Firefox, Safari, Edge)
- Kiểm tra tốc độ tải trang
- Test accessibility (a11y)

#### **5️⃣ Giai Đoạn Triển Khai**
- Push code lên GitHub
- Cấu hình GitHub Pages hoặc deployment khác
